# think-oms
基于DDD四层架构的订单中台，此文只是一篇DEMO 供领域驱动设计实践参考

# 背景
设计一款可以同时支持接入自有平台，拼多多，抖音，淘宝天猫、小米、小红书、快手等平台的订单，做统一管理用于下游结算，履约，售后，安装等服务
